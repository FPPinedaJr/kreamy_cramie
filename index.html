Example 1:

Let‚Äôs solve this system of equations:

2x + 3y = 5

x - y = 1


Solution:

Step 1: Write the coefficient matrix and constant matrix:

                Coefficient matrix:
                | 2  3 |
                | 1 -1 |
        
                Constant matrix:
                |  5  |
                |  1  |

Step 2: Calculate the determinant of the coefficient matrix (D):

                D = (2 * -1) - (3 * 1) = -2 - 3 = -5

Step 3: Replace the first column of the coefficient matrix with the constants and calculate Dx:

                Dx matrix:
                | 5  3 |
                | 1 -1 |
        
                Dx = (5 * -1) - (3 * 1) = -5 - 3 = -8

Step 4: Replace the second column of the coefficient matrix with the constants and calculate Dy:

                Dy matrix:
                | 2  5 |
                | 1  1 |
        
                Dy = (2 * 1) - (5 * 1) = 2 - 5 = -3

Step 5: Solve for x and y:

                x = Dx / D = -8 / -5 = 1.6
                y = Dy / D = -3 / -5 = 0.6

Solution: x = 1.6, y = 0.6

Solution:

                Dx = -70, Dy = 65, Dz = 30
